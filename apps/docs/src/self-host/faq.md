# 常见问题

## 访问页面后无法登录怎么办？

先确认至少配置了一种登录方式。如果使用 OAuth，请检查客户端 ID、客户端密钥、回调地址和反向代理后的公开域名是否一致。

## AI 对话无法生成怎么办？

检查以下项目：

- 是否已配置模型服务商。
- 是否已同步或手动添加模型。
- 用户是否选择了默认模型。
- 服务商 API Key 是否有效。
- `agent` 服务是否健康。
- Redis 是否可用。

## 文档协作连接失败怎么办？

检查：

- `collab` 服务是否健康。
- WebSocket 是否被反向代理正确转发。
- 当前用户是否拥有编辑权限。
- 文档是否已进入回收站。
- `API_INTERNAL_URL` 是否指向 API 服务 origin。

## 图片或附件无法显示怎么办？

检查对象存储：

- `STORAGE_ENDPOINT` 是否可达。
- 访问密钥是否正确。
- RustFS 或 S3 服务是否健康。
- 反向代理是否允许资源请求通过。

## 能否关闭后台公网访问？

可以。建议只把 Web 正常访问入口暴露到公网，把数据库、Redis、对象存储控制台和内部服务端口限制在内网。
