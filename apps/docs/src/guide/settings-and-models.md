# 个人设置与模型

个人设置用于维护当前账号、偏好和 AI 模型使用方式。

## 账号

账号设置包含基础资料、邮箱绑定、密码修改和 OAuth 绑定。具体可用登录方式取决于管理员在实例后台启用的能力。

## 偏好

偏好设置用于调整当前用户的界面语言、外观等本地体验。部分偏好只影响当前用户，不会改变其他成员看到的内容。

## 服务商

服务商用于接入 AI 模型。当前支持平台预设服务商，也支持 OpenAI-Compatible、Anthropic-Compatible 等兼容服务商。

添加服务商后，需要配置 API 地址、认证方式、密钥和模型列表。

## 默认模型

默认模型用于决定不同 AI 入口使用哪个模型。当前主要包含聊天默认模型和文档生成/改写默认模型。

如果某个入口没有可用默认模型，界面会提示先完成模型配置。

## 与自部署后台管理的区别

个人设置只影响当前用户或当前用户添加的服务商。自部署后台管理中的模型服务商面向整个实例，需要系统管理员权限。
