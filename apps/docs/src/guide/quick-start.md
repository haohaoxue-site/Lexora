# 快速开始

这页面向已经可以访问 Lexora 实例的用户。如果你要部署自己的实例，请先阅读 [Docker Compose 部署](/self-host/docker-compose)。

## 1. 登录账号

打开实例地址后，使用已启用的登录方式进入系统。自部署实例的登录方式由管理员在后台配置，可能包含账号密码、GitHub、Linux Do 或 Google OAuth。

## 2. 配置模型

进入 `设置 -> 服务商`，添加或启用可用的 AI 服务商。服务商可以来自平台预设，也可以是 OpenAI-Compatible 或 Anthropic-Compatible 地址。

进入 `设置 -> 默认模型` 后，为聊天和文档 AI 选择默认模型。未配置默认模型时，AI 对话和文档生成入口可能无法直接运行。

## 3. 开始 AI 对话

从左侧导航进入 `对话`。你可以创建会话、选择模型并发送问题。对话支持流式生成、停止生成、失败重试和消息分支切换。

## 4. 创建文档

进入 `文档` 后创建页面。文档页面使用富文本编辑器，适合写笔记、说明文、项目资料和协作内容。

## 5. 邀请协作

在文档协作入口中，可以按用户码邀请指定用户，也可以生成协作链接。协作权限分为可阅读和可编辑，授权范围可以仅当前页面，也可以覆盖子页面。

## 6. 发布公开页面

如果文档需要面向外部访客展示，可以进入发布设置。Lexora 支持单页发布和站点发布。发布页面是只读访问，不会让访客进入协作编辑。
