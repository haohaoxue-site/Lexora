# 公开发布

公开发布用于把文档展示给外部访客。发布内容只读，不进入文档协作运行时。

## 单页发布

单页发布使用 `/p/:documentId` 访问。文档可以显式开启或关闭公开状态，也可以继承父页面的公开状态。

单页发布适合展示一篇说明文、公告、教程或临时资料。

## 站点发布

站点发布使用 `/s/:siteId` 和 `/s/:siteId/:documentId` 访问。站点可以配置标题、图标、首页、侧边栏页面、分组和顶部导航。

站点发布适合把一组文档组织成类似 VitePress 的公开资料站。

## 资源访问

公开发布会使用专用的公开资源访问范围。发布页面不会复用登录用户在 `/docs` 中读取图片或附件的权限。

## 内链处理

发布正文中的内部链接只会跳转到已经公开的目标文档。未公开目标会保留链接视觉，但不会暴露未发布页面。
