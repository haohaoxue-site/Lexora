# User Management

User management is used to view and maintain accounts in the instance.

## User List

The admin user page displays instance users. Administrators can filter by keyword and inspect identity, account status, and related data summaries.

## User Identity

SamePage AI uses a global `User` identity. Collaboration invites use user codes instead of public usernames.

Display names are editable and must not be treated as unique identity credentials.

## Registration and Invites

The instance can control registration through admin login governance. Open registration, invite codes, and third-party login availability depend on admin configuration.

## Risky Operations

Changing user status, login methods, or account data can affect access. Confirm the target account and impact before performing risky actions.
